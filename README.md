# Proyecto Java – Facultad

Este proyecto fue desarrollado como parte de la materia Programación II en la universidad.  
Implementa conceptos de **Programación Orientada a Objetos (POO)** en Java.

## Tecnologías
- Java
- MySQL (para persistencia)
- MySQL Workbench

## Funcionalidades
- Clases con herencia e interfaces
- Manejo de excepciones personalizadas
- CRUD básico de entidades
- Ejemplos de colecciones y polimorfismo

## Objetivo académico
- Practicar fundamentos de POO
- Integrar Java con base de datos
- Aplicar buenas prácticas de programación
