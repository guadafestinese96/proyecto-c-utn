# Proyecto en C – Facultad

Este proyecto fue desarrollado como parte de la materia de Programación en C en la universidad.  
Implementa conceptos de **programación estructurada**, manejo de memoria y estructuras de datos.

## Tecnologías
- Lenguaje C
- Compilador GCC

## Funcionalidades
- Manejo de arrays y estructuras
- Funciones y modularización
- Ejercicios de lógica y algoritmos básicos

## Objetivo académico
- Practicar fundamentos de programación estructurada
- Comprender el uso de punteros y memoria dinámica
- Aplicar buenas prácticas en C
